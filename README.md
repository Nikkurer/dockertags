# dockertags 
`dockertags` get docker image tags from [Docker Hub](https://hub.docker.com) uses API v1.

# Usage
```bash
$ dockertags -h
    usage: dockertags.py [-h] image
    
    Get docker image tags.
    
    positional arguments:
      image       Image name
    
    optional arguments:
      -h, --help  show this help message and exit
```
